Qu es JavaScript?

Es un lenguaje de programacin creado en 1995 por Brendan Eich, para darle dinamismo e interaccin a los sitios web.

Hoy es un lenguaje de propsito general.

Por qu JavaScript?

  • Es el lenguaje que entienden y ejecutan los navegadores (browsers).
  • Es relativamente fcil de aprender, por lo que es una buena opcin como primer lenguaje.
  • Hay mucha demanda de desarrolladores JavaScript.
  • Ahora tambin podemos ejecutar cdigo JavaScript fuera del browser, usando NodeJS.

Variables

Es una caja para guardar valores, de diferentes tipos. Esta caja tiene una etiqueta, el nombre de la variable.

Son reutilizables: una vez creadas, podemos usarlas todas las veces que querramos.

Se pueden reasignar, cambiar su valor. Por eso se llaman variables.

Declarar una variable

Declarar es crear una nueva variable.

Usamos la palabra var para crear nuevas variables.

var nombreDeLaVariable;

Nota: las sentencias en JS terminan con ;

Importante: prob todo el cdigo en tu consola!

Constantes

Son espacios con nombre para guardar valores (tambin son cajas con etiquetas, como las variables) que no varan.

Son reutilizables, pero no se pueden reasignar.

Si reasignamos una constante, obtenemos un error

const saludo = 'Hola!';
saludo = 'Hola Mundo!';
// => Uncaught TypeError: Assignment to constant variable

Tipos de datos

Podemos usar valores de diferentes tipos en JavaScript.

Tipos principales en JS

  • String (Cadena de caracteres)
  • Number (Nmeros)
  • Boolean (Booleanos)

String

Representan cadenas de caracteres (texto).

Se escriben entre comillas simples dobles, sin mezclar.

Cualquier valor entre comillas es un String, por ejemplo, un nmero.

OR ||

Retorna true si al menos una de las expresiones es verdadera; sino, retorna false.

true || true
// => true
true || false
// => true
false || false
// => false

NOT !

Negacin. Retorna false si la expresin es verdadera; sino, retorna true.

// En este caso poner !(true) o bien !true es lo mismo
!(true)
// => false
!(2 > 1)  // false, porque 2 > 1 es true
!(/* expresin booleana verdadera */)  // false
!(/* expresin booleana falsa */)  // true

Valores falsos (falsy values)

Son aquellos valores que se interpretan como falsos si se evalan en el contexto de una expresin booleana.

  • false
  • null
  • undefined
  • ''
  • 0
  • NaN

Valores verdaderos (truthy values)

Todos los valores que no se encuentren en la lista anterior, se consideran verdaderos.

Ciclos

Nos permiten iterar sobre diferentes colecciones de valores

Tambin iterar hasta que ciertas condiciones se cumplan o no

While

Se ejecuta hasta que la condicin booleana sea falsa.

var i = 0;
// Iteracin N 0 hasta Iteracin N 4
while (i < 5) {
  console.log('Iteracin N ' + i);
  i++;
}

For

Se ejecuta hasta que la condicin booleana sea falsa y consiste en tres expresiones opcionales.

for (expresin inicial; condicin; expresin final) {
  ...
};
  • expresin inicial: declaramos una variable como contador.
  • condicin: expresin evaluada antes de cada iteracin.
  • expresin final: expresin evaluada al final de cada iteracin.

Ejemplo I

for (var i = 0; i < 10; i++) {
  console.log('Iteracin N ' + i);
}

Ejemplo II

var animales = ['Perro', 'Gato', 'Tortuga'];
 
// Recorremos el array animales y mostramos cada uno
for (var i = 0; i < animales.length; i++) {
  console.log('Las mascotas disponibles son: ' + animales[i]);
}
